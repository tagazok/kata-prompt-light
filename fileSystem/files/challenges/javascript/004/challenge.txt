**PALINDROME**

A **palindrome** is a word, number, phrase, or other sequence of symbols that reads the same backwards as forwards, such as *madam* or *racecar*, the date and time *12/21/33 12:21*.

Example of function call:
```javascript
pal("madam");
// => true
```
